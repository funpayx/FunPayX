# FunPayX

**FunPayX** - телеграм-бот для автоматизации работы с FunPay.
Приём и обработка заказов, отзывов, сообщений, мониторинг лотов.

## Возможности

- Автоматическая обработка заказов - приём, подтверждение, выдача товара
- Работа с отзывами - автоматические ответы на отзывы
- Обработка сообщений - автоответы, поддержка сценариев
- Плагинная система - расширение функционала без изменения ядра

## Быстрый старт

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Создайте файл `.env` в funpayx/ и укажите переменные:
   ```env
   BOT_TOKEN=<токен телеграм-бота>
   GKEY=<ваш GoldenKey из FunPay>
   GSEAL=<ваш GoldenSeal из FunPay>
   ```

3. Запустите:
   ```bash
   python funpayx
   ```

Бот запускает два компонента параллельно:
- Telegram-бот (aiogram) - интерфейс управления
- FunPay-слушатель (fpx) - обработка событий FunPay

## Плагины

FunPayX поддерживает плагины - можно расширять функциональность, не трогая ядро.
Разработка плагина сводится к созданию папки в `plugins/` и написанию пары хендлеров.

Подробная документация по созданию плагинов - в [docs/plugins.md](docs/plugins.md).

## Технологии

- Python 3.10+
- aiogram 3.x - Telegram Bot API
- fpx - фреймворк для фанпей
- SQLAlchemy + SQLite - хранение данных
- asyncio - конкурентность
